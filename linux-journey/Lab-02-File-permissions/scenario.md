# Lab 02 - File Permissions & Security Incident

## Scenario

While working as a Linux Administrator at PayFlow Inc., a security issue was reported.

A member of the engineering team was able to access and modify files in the finance directory.

This violates the company's access control policy.

You were tasked to:
- Investigate the issue
- Fix the permissions
- Ensure only authorized users can access their department folders
- Enforce proper group ownership for all files going forward
